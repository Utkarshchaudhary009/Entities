### Typography

#### Primary Font: **Suisse Intercirc** (or similar geometric sans-serif with character)

**Why:** Distinctive, modern, premium feel. Not overused like Inter. Excellent for headlines and creates instant brand recognition.

#### Fallback Stack: **Neue Haas Grotesk**, **Akzidenz-Grotesk**, system sans-serif

#### Font Hierarchy

| Element | Weight | Size | Line Height | Notes |
|---------|--------|------|-------------|-------|
| **H1 - Hero** | Bold (700) | 48px / 3rem | 1.1 | Tight tracking (-0.02em) |
| **H2 - Section** | Semibold (600) | 32px / 2rem | 1.2 | |
| **H3 - Card Title** | Medium (500) | 20px / 1.25rem | 1.3 | |
| **Body** | Regular (400) | 16px / 1rem | 1.6 | Generous line-height |
| **Caption** | Regular (400) | 14px / 0.875rem | 1.5 | |
| **Button** | Semibold (600) | 14px / 0.875rem | 1 | ALL CAPS optional |
| **Price** | Bold (700) | 18px / 1.125rem | 1 | Tabular figures |
| **Tagline** | Medium (500) | 24px / 1.5rem | 1.4 | Italic for emphasis |

#### Typography Rules

- **Headlines**: Tight tracking (-0.02em), bold weight, no more than 6 words when possible
- **Body**: Generous line-height for readability, max-width 65ch
- **All caps**: Only for badges, labels, category tags, and CTAs
- **Numbers**: Use tabular figures for prices and data
- **Never center-align body text**
- **One typeface, endless expression** — use weight and size contrast, not multiple fonts

## Photography Direction

### Style Principles

1. **Real people, real moments** — Authentic street style, not studio artificial
2. **Urban India first** — Indian streets, architecture, local culture, then global
3. **Night as signature** — Urban nightlife, neon, street lights, moody atmospheres
4. **Natural lighting preferred** — Golden hour, overcast, street lights
5. **Movement and energy** — Candid shots, motion blur, dynamic poses
6. **Imperfection as aesthetic** — Slightly underexposed, raw grain, authentic edge
7. **Diverse representation** — Different body types, skin tones, styles, ages

### Shot Types

| Shot Type | Ratio | Usage |
|-----------|-------|-------|
| Hero Banner | 16:9 | Homepage hero, campaign pages |
| Product Card | 3:4 | Shop grid, category pages |
| Product Detail | 1:1 | Main product images |
| Lifestyle | 4:5 | Instagram, lookbooks |
| Thumbnail | 1:1 | Cart, admin, search |
| Editorial | 3:2 | Lookbooks, press |

### Image Treatment

- **Vignette**: Subtle darkening at edges (hero images only)
- **Contrast**: Slightly elevated for punch (+10-15%)
- **Saturation**: Natural, slightly desaturated for premium feel
- **Grain**: Subtle film grain (ISO 800-1600 equivalent) for mood
- **Color grading**: Cool shadows, warm highlights (cinematic)
- **Shadows**: Lift slightly — never crush to pure black in lifestyle shots

### What to Avoid

- ❌ Over-posed, catalog-style photos
- ❌ White studio backgrounds
- ❌ Over-retouched skin
- ❌ Stock photo aesthetics
- ❌ Cluttered backgrounds
- ❌ Perfect lighting — embrace the raw
- ❌ Models who look like models — real people only

## UI Principles

### Design Philosophy

**"Premium simplicity"** — Every element earns its place. Nothing extra. Nothing missing. Nothing ordinary.

### Spacing System

Base unit: **4px**

| Size | Value | Usage |
|------|-------|-------|
| xs | 4px | Tight spacing, icon gaps |
| sm | 8px | Small component padding |
| md | 16px | Default spacing |
| lg | 24px | Section gaps |
| xl | 32px | Major sections |
| 2xl | 48px | Page sections |
| 3xl | 64px | Hero spacing |
| 4xl | 96px | Major page breaks |

### Border Radius

| Element | Radius | Notes |
|---------|--------|-------|
| Buttons | 8px | Never round buttons |
| Cards | 12px | |
| Images | 8px | |
| Inputs | 6px | |
| Modals | 16px | |
| Tags/Badges | 4px | Sharp, not pill-shaped |
| Product images | 4px | Subtle, almost imperceptible |

### Shadows

```css
--shadow-sm: 0 1px 2px rgba(0, 0, 0, 0.1);
--shadow-md: 0 4px 6px rgba(0, 0, 0, 0.15);
--shadow-lg: 0 10px 25px rgba(0, 0, 0, 0.2);
--shadow-xl: 0 20px 40px rgba(0, 0, 0, 0.25);
--shadow-card: 0 4px 20px rgba(0, 0, 0, 0.3); /* Premium card shadow */
```

### Animation Principles

| Type | Duration | Easing |
|------|----------|--------|
| Micro-interactions | 150ms | ease-out |
| Hover states | 200ms | ease-in-out |
| Page transitions | 300ms | ease-in-out |
| Modals/overlays | 250ms | ease-out |
| Skeleton pulse | 1.5s | ease-in-out (loop) |
| Product reveal | 400ms | cubic-bezier(0.16, 1, 0.3, 1) |

### Motion Philosophy

- **Purposeful**: Animation guides attention, not distracts
- **Quick**: Nothing should take longer than 300ms (except reveal animations)
- **Natural**: Ease curves feel organic
- **Reducible**: Respect `prefers-reduced-motion`
- **Snappy**: Premium feels responsive, never sluggish

## Component Guidelines

### Buttons

#### Primary CTA

- Background: Entity Crimson `#C41E3A`
- Text: White
- Hover: Darken 15% or subtle glow
- Active: Scale to 98%
- Disabled: 50% opacity, no interaction
- Padding: 14px 28px
- Text: ALL CAPS, letter-spacing: 0.05em

#### Secondary Button

- Background: Transparent
- Border: 1px solid white (or `#FAFAFA`)
- Text: White
- Hover: White background, Void Black text
- Padding: 14px 28px

#### Ghost Button

- Background: Transparent
- Text: Ash `#6B6B6B`
- Hover: Text becomes white
- No border

### Cards

- Background: Carbon `#1A1A1A`
- Border: None (use shadow for elevation)
- Hover: Subtle lift + shadow increase (`--shadow-card`)
- Image: Always fills top portion
- Content padding: 16px
- Price: Entity Crimson on hover

### Forms

- Input background: Carbon `#1A1A1A`
- Border: 1px solid Slate `#2A2A2A`
- Focus: Entity Crimson border `#C41E3A`
- Error: Error Red border + message below
- Labels: Above input, 14px, Medium weight
- Placeholder: Ash `#6B6B6B`, italic

### Navigation

- Background: Void Black with blur (backdrop-filter: blur(20px))
- Mobile: Full-width hamburger menu, slide from right
- Active state: Entity Crimson underline
- Hover: Fade transition, slight text brightening

## Trust Architecture

For a new premium brand, trust is everything. Build it into every touchpoint.

### Product Pages Must Include

- **Size guide with photos** — Real people wearing each size
- **Material close-ups** — Show the GSM quality, stitch detail, fabric texture
- **"As worn by" gallery** — Customer photos, not just models
- **Fit predictor** — "You're a size M in our Oversized Tee" based on past purchases
- **Transparent returns** — "Free returns within 14 days. No questions."
- **Honest stock levels** — Real numbers, not fake scarcity

### Trust Signals Throughout Site

- **Reviews with photos** — Customer photos carry more weight than model shots
- **Press mentions** — "As seen in [Publication]" with logos
- **Founder story** — Real face, real name, real story on About page
- **Instagram feed** — Live feed of community wearing the brand
- **Response time** — "We reply within 2 hours" on contact page

### Copywriting Guidelines

| Section | Tone | Example |
|---------|------|---------|
| Product names | Bold, minimal | "Oversized Tee - Void", "Cargo Jogger - Olive" |

- Use evocative, abstract names
- Reference culture, music, art, or emotion
- Never use generic names like "Summer Collection"
- Examples: "After Hours", "Neon Dreams", "Archive 01", "Streets of Mumbai", "Midnight Protocol"

## Brand Touchpoints

### Packaging

- **Box**: Matte black mailer box with blind embossed logo (no ink, just impression)
- **Texture**: Soft-touch matte finish (feels premium before opening)
- **Tissue**: Black tissue paper with subtle pattern
- **The Drop Card**: Collectible card with product info
- **Thank you note**: Handwritten-style, signed by the team
- **Size/fit insert**: Minimal, useful, not promotional

### Unboxing Experience

1. Weight and texture of box signals quality immediately
2. Black tissue paper reveal with subtle brand pattern
3. Product folded with intention, not stuffed
4. The Drop Card on top — collectible, shareable
5. Unexpected detail: sticker pack or pin in first order
6. Easy returns info tucked at bottom (transparent, not hidden)

### Sensory Details

- **Smell**: Fresh fabric, no chemical odor
- **Touch**: Soft-touch box, quality tissue, premium product feel
- **Sound**: Box has weight, tissue crinkles softly
- **Sight**: Everything is black, white, crimson — cohesive
